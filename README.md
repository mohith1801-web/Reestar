Reestar is a simple and modern restaurant website created using HTML and CSS. This website displays different food items with images, descriptions, and an order button. It also includes a contact form where users can enter their details.

FEATURES:
* Clean and simple user interface
* Navigation bar with menu options
* Food items with images and descriptions
* Order Now buttons
* Contact form section
* Dark theme design
* Flexbox layout for proper alignment

TECHNOLOGIES USED:

* HTML5 for structure
* CSS3 for styling
* Google Fonts for typography
* Flexbox for layout

PROJECT STRUCTURE:

* index.html (Main HTML file)
* style.css (CSS styling file)
* img folder (Contains food images)
* README.md (Project description)

PURPOSE:
This project was created to practice basic web development skills including HTML layout, CSS styling, Flexbox, and form creation.
